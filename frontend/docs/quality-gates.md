# Quality Gates

用于每个阶段结束时执行的最小回归清单。

## 1. 鉴权与路由

- 未登录访问业务路由应跳转到 `/login`
- 登录后首次跳转应加载用户信息与动态菜单路由
- 无权限访问受控页面应跳转 `/403`
- 登出后 token 与用户态应被清理

## 2. 布局与偏好

- 侧边栏折叠状态刷新后保持
- 主题模式与主题色刷新后保持
- 语言切换刷新后保持
- 标签页新增、关闭、关闭其他正常

## 3. 业务主线

- 用户、角色、菜单、权限、部门、租户、日志页面可打开
- 页面首屏加载失败时有错误提示
- 分页列表页可正常翻页

## 4. Playground 能力

- 权限示例可展示当前角色/权限
- 主题示例可切换深浅色并修改主题色
- 表格、表单、缓存示例可正常交互

## 5. 工程质量

- `pnpm type-check` 通过
- `pnpm lint` 通过（允许历史遗留告警，但不新增错误）
